    Discord.js Example Bot

    About

    This is a sample repository for a Discord Bot using Discord.js.

    • 日本語 (Japanese)
    • 简体中文 (Chinese)

    Usage

    1. Install dependencies

    On Windows, run the following command in Command Prompt:

    setup.bat

    Or, if Bun is already installed:

    bun i

    2. Run the bot

    bun run src/index.ts

    3. Update the bot

    bun run update

    Or run the /update command in Discord (requires admin permissions in the bot config).
