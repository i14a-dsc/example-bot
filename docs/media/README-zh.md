## 关于

这是一个使用 [Discord.js](https://discord.js.org/) 的 Discord 机器人示例仓库。

- [English](README.md)
- [日本語 (Japanese)](README-ja.md)

## 使用方法

### 1. 安装依赖

在 Windows 上，请在命令提示符中运行以下命令：

```
setup.bat
```

或者，如果已安装 Bun：

```
bun i
```

### 2. 启动机器人

```
bun run src/index.ts
```

### 3. 更新机器人

```
bun run update
```

或者在 Discord 上运行 `/update` 命令（需要在机器人配置中拥有管理员权限）。
